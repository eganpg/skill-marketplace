Pete only

Review skill ideas, start builds, and manage roadmap status.

How the workflow works

  1. 1. Review ideas below. When one is ready to build, click "Start Building".
  2. 2. A pre-filled GitHub Issue opens with all the context Claude needs to build the skill. Submit it.
  3. 3. Use Claude (with the issue context) to generate the .skill file and README.
  4. 4. Open a PR to add the skill to the marketplace. Update roadmap.json status to done.
