# Agents

Agent definitions and workflow configurations for Claude and Copilot. Patterns for building AI that does more than answer questions.

## Structure

- [`claude/`](./claude/) — Sub-agents, workflows, and MCP configs for Claude
- [`copilot/`](./copilot/) — Custom agents and agent mode configs for Copilot
- [`antigravity/`](./antigravity/) — Custom agents and agent mode configs for Antigravity

Browse each subdirectory for individual items with usage instructions and examples.
