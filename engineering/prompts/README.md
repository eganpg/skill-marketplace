# Prompts

Reusable system prompts, task prompts, and multi-step prompt chains for Claude and Copilot. Drop them into your workflow or adapt them to your needs.

## Structure

- [`claude/`](./claude/) — System prompts, task prompts, and prompt chains for Claude
- [`copilot/`](./copilot/) — Instruction files and chat prompt patterns for Copilot

Browse each subdirectory for individual items with usage instructions and examples.
