# QASP Report Generator

**Category:** Document Processing
**Author:** Pete Egan
**Version:** 1.0.0
**Tags:** program-manager, qasp, contracts, reporting, government

---

## What it does

Generates a structured QASP (Quality Assurance Surveillance Plan) submission based on contractor delivery reports. Includes a reference template and QA standards to ensure outputs are consistent, complete, and ready to submit.

---

## Who it's for

Program managers responsible for documenting and reporting contractor performance against QASP metrics and standards.

---

## How to use it

1. Open Claude with this skill installed
2. Provide your contractor's delivery report (paste the text or attach the document)
3. The skill will generate a complete QASP submission using the included reference template and QA standards
4. Review and submit the output

---

## What it produces

- A completed QASP submission document aligned to the reference template
- Performance assessment against defined QA standards
- Ready-to-submit formatted output
