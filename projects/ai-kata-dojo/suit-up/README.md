# Stage 1 -- Suit Up

Time: 3 minutes

Pick one AI tool. Either something you want to get better at or something you have never tried. You will use this tool for the entire session.

## Your Options

### CLI Tools

Command-line interfaces. You type prompts in your terminal and the AI reads and edits your code directly.

| Tool | How to Get It |
|------|--------------|
| Claude Code | `npm install -g @anthropic-ai/claude-code` then run `claude` |
| Gemini CLI | `npm install -g @google/gemini-cli` -- see Google's install docs |
| Copilot CLI | `gh extension install github/gh-copilot` |
| Codex CLI | `npm install -g @openai/codex` |
| OpenCode | `go install github.com/opencode-ai/opencode@latest` |
| Aider | `pip install aider-chat` |

### IDE Tools

AI built into your editor. Code suggestions, inline chat, and agent panels.

| Tool | How to Get It |
|------|--------------|
| VS Code + Copilot | Install the GitHub Copilot extension |
| Cursor | Download from cursor.com -- it is a fork of VS Code |
| Windsurf | Download from windsurf.com |

### Desktop / Chat Apps

Conversational interfaces. You paste code or describe problems and get responses.

| Tool | How to Get It |
|------|--------------|
| Claude Desktop | claude.ai or download the desktop app |
| ChatGPT | chatgpt.com or download the desktop app |
| Gemini | gemini.google.com |
| Codex | codex.openai.com |

## What to Do Right Now

1. Pick one tool from any category.
2. Open it. Make sure it is working.
3. If you need to install something, do it now.
4. If you already have your tool ready, just sit tight.

Do not overthink this. There is no wrong choice. You are going to practice the same skills regardless.

## If You Are Not Sure What to Pick

- If you have never used any AI coding tool: start with your IDE tool (Copilot in VS Code or Cursor). Lowest friction.
- If you already use Copilot daily: try a CLI tool like Claude Code or Gemini CLI. Different paradigm.
- If you want the most autonomous experience: try Claude Code or Cursor in agent mode.
- If you cannot install anything on your machine: use a desktop app like Claude or ChatGPT.
