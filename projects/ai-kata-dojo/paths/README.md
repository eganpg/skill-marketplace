# Stage 4 -- Choose Your Path

Time: 10 minutes (or unlimited if you are solo)

Pick one topic that interests you. Open its guide. Follow the exercises. Each path has a concrete goal -- by the end, you will have built or achieved something real, not just read about it.

If you finish early, pick another.

## The Paths

### Getting Started

| Path | You Will Build / Achieve |
|------|-------------------------|
| [Prompt Craft](./prompt-craft/) | Write a prompt that works on the first try using the specificity ladder |
| [Context Management](./context-management/) | Audit your tool's context and solve a task with minimal, targeted input |
| [Error Recovery](./error-recovery/) | Fix an AI mistake surgically without starting over |

### Daily Workflows

| Path | You Will Build / Achieve |
|------|-------------------------|
| [Custom Instructions](./custom-instructions/) | Create a config file that makes every session start smarter |
| [Multiple Sessions](./multiple-sessions/) | Switch between two tasks and resume without losing context |
| [Code Review](./code-review/) | Run an AI-assisted review on a real PR across 4 dimensions |
| [Daily Summaries](./daily-summaries/) | Generate today's standup summary and a repeatable process for tomorrow |
| [Notes and Memory](./notes-and-memory/) | Set up persistent memory so your tool remembers you across sessions |

### Power User

| Path | You Will Build / Achieve |
|------|-------------------------|
| [Autonomy](./autonomy/) | Let AI plan and execute a multi-step task with scoped boundaries |
| [Integrations](./integrations/) | Connect your tool to GitHub, Slack, or another service via CLI or MCP |
| [Debugging](./debugging/) | Diagnose a bug using hypothesis-driven debugging with AI |
| [Testing](./testing/) | Write tests using TDD with AI and verify they catch real bugs |
| [Large Project Orchestration](./large-project-orchestration/) | Break a big task into an AI-executable plan and complete the first step |

### Creative

| Path | You Will Build / Achieve |
|------|-------------------------|
| [Slides Generation](./slides-generation/) | Generate a real 5-slide presentation you can actually present |
| [Image Generation](./image-generation/) | Create a diagram, mockup, or visual you can use in your work |

## Not Sure Where to Start?

- Never used AI tools before: **Prompt Craft**
- Use AI daily, want to level up: **Custom Instructions** or **Daily Summaries**
- Want to see the future: **Autonomy** or **Integrations**
- Skeptical about AI: **Error Recovery**
- Need something right now: **Slides Generation** or **Daily Summaries**
- Have a big project looming: **Large Project Orchestration**
