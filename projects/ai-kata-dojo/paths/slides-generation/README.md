# Path -- Slides Generation

By the end of this path, you will have generated a real slide deck using your AI tool.

## The Goal

Create a 5-slide presentation on any topic you choose. The slides should be ready to present -- not a rough draft.

## Exercise 1 -- Generate a Deck (5 min)

Pick a topic you actually need to present on (or use "What I learned at the AI Dojo today").

Ask your AI tool:

> "Create a 5-slide presentation on [topic]. For each slide, give me: a title, 3-4 bullet points, and speaker notes. Keep it concise -- no slide should have more than 20 words in bullets."

Review the output. Then refine:

> "Slide 3 is too dense. Split it into two slides. And make the opening slide more attention-grabbing."

## Exercise 2 -- Generate the Actual File (5 min)

Now turn it into a real file. Depending on your tool:

**Option A -- Markdown slides** (works in any tool):
> "Convert these slides into a Marp-compatible markdown file I can present from."

**Option B -- HTML slides** (works in any tool):
> "Create an HTML file using reveal.js that I can open in a browser and present."

**Option C -- Google Slides / PowerPoint** (requires integration):
> "Create this as a Google Slides presentation" (if your tool has Google Slides MCP)

Open the file and verify it works.

## Exercise 3 -- Iterate on Design (5 min)

The first version will look generic. Practice directing the visual style:

> "Make the slides more minimal. Dark background, white text, one key phrase per slide instead of bullets. I want this to feel like a keynote, not a textbook."

Or:

> "Add a simple diagram to slide 2 showing the workflow. Use ASCII art or an SVG."

The skill here is knowing that you can iterate on presentation design the same way you iterate on code.

## What You Built

A real slide deck you can actually use. If the topic was "What I learned at the AI Dojo today," you now have your Stage 5 share-out ready.

## Going Deeper

- Generate slides from an existing document (paste your notes, get slides out)
- Try generating slides with embedded code examples that are syntax-highlighted
- Experiment with different presentation frameworks (Marp, reveal.js, Slidev)
- Ask the AI to create a presentation outline first, review it, then generate slides from the approved outline
